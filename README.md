📊 Data Analytics Project

📌 Overview
This project demonstrates an end-to-end data analytics workflow, starting from data loading and exploration to visualization and reporting. The objective is to extract meaningful insights from raw data using industry-relevant tools and techniques.

📂 Dataset
The dataset used in this project contains structured data relevant to analysis.
It includes multiple features (columns) used for data cleaning, transformation, and visualization.
Source: Kaggle
🛠️ Tools & Technologies
Python – Data loading, preprocessing, and analysis
Pandas, NumPy – Data manipulation
Matplotlib / Seaborn – Data visualization
MySQL – Querying and data extraction
Power BI – Dashboard creation and visualization
Jupyter Notebook – Development environment.

🔄 Project Steps
1. Data Loading
Imported dataset using Python (Pandas)
Checked data structure, types, and initial records
2. Exploratory Data Analysis (EDA)
Analyzed data distributions and patterns
Identified trends, correlations, and anomalies
Visualized key insights using plots and charts
3. Data Cleaning
Handled missing values
Removed duplicates
Standardized and formatted data for consistency
4. SQL Analysis (MySQL)
Imported cleaned data into MySQL
Executed SQL queries to extract insights
Performed aggregations, filtering, and joins
5. Dashboard Creation (Power BI)
Connected Power BI to MySQL / dataset
Built interactive dashboards
Created KPIs, charts, and filters for better insights
6. Reporting
Summarized key findings
Highlighted trends and business insights
Presented results in a structured format
